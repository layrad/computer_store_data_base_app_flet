# DNS 2 - Кроссплатформенное клиентское приложение (Flet Client)

Кроссплатформенное клиентское приложение для ОС Windows и Android, предназначенное для взаимодействия с базой данных торговых операций и построения глубоких аналитических бизнес-отчетов.

## Установка и запуск

Для работы с приложением не требуется ручная установка зависимостей. Достаточно скачать готовый дистрибутив последней версии из раздела релизов (Releases) на GitHub для вашей операционной системы:

* Для настольных компьютеров на ОС Windows: скачайте архив ZIP, распакуйте его в любую удобную папку и запустите исполняемый файл приложения.
* Для мобильных устройств на ОС Android: скачайте APK-пакет и установите его на мобильный телефон.

Для подключения к вашему запущенному серверу FastAPI перед запуском приложения укажите его сетевой адрес в окне настроек подключения.

## Руководство по интерфейсу

Переход между разделами осуществляется через левое выдвижное навигационное меню.

### Экран «База» (Управление данными)

* Поисковая строка вверху экрана - позволяет осуществлять умный семантический поиск комплектующих по смыслу описания и характеристик.
* Вкладка «Товары» - содержит таблицу всех комплектующих. Сортировка выполняется кликом по заголовкам колонок. Иконка карандаша в строке открывает диалог редактирования товара, иконка корзины - помечает товар как удаленный.
* Вкладка «Продажи» - отображает реестр всех торговых транзакций и чеков с суммами, статусами и именами продавцов.
* Кнопка «Добавить товар» - открывает модальное окно добавления комплектующего с автоматической генерацией SKU по маске категории.
* Кнопка «Добавить продажу» - открывает окно регистрации транзакции с авторасчетом итоговой стоимости с учетом скидок и автоматическим списанием остатков со склада.
* Кнопка «Сохранить изменения» - осуществляет массовую асинхронную отправку всех локальных правок (добавления, удаления, редактирования) на сервер.

### Экран «Отчеты» (Бизнес-аналитика)

* Выпадающий список типов отчетов - позволяет моментально выбрать и рассчитать один из 15 встроенных аналитических срезов (включая ABC-анализ выручки, XYZ-анализ стабильности спроса, рентабельность брендов, риски возвратов).
* Текстовое поле «Свой вариант...» - активируется при выборе ручного режима и позволяет написать ИИ-запрос на естественном русском языке.
* Кнопка «Составить отчет» - запускает процесс локального расчета или отправки NLP-запроса на сервер.
* Кнопки «Сохранить как TXT» и «Сохранить как PDF» - экспортируют сформированный табличный отчет в структурированные локальные файлы на диск.

### Экран «О нас» (Информационный)

* Содержит имя автора проекта и две активные кнопки для быстрого перехода на публичные GitHub-репозитории серверной и клиентской частей.
